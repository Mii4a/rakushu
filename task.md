# Task

## Title
主要ナビゲーション配下（求人一覧 / 保存した求人 / 比較 / 履歴書 / 判断基準 / 料金 / 設定）の server-side application error を調査・修正する

## Goal
対象ルートで発生しているサーバーサイドのアプリケーションエラーを再現し、根本原因を特定し、必要最小限の修正で各ページが正常表示される状態に戻す。

## Scope for this task
1. 症状の再現と対象ルートの確認
2. ログ・ビルド・本番相当実行でのエラー特定
3. 原因コードの切り分け
4. 必要な修正実装
5. 対象ルートの再検証

## Non-goals
- 今回のエラーと無関係な大規模リファクタ
- UI 改善や文言調整のみの変更
- 別機能の追加実装

## Deliverables
- 必要なコード修正
- 更新された `task.md`
- 更新された `implementation_plan.md`
- 更新された `walkthrough.md`
- 再現手順・原因・対策・検証結果の共有

## Completion criteria
- 対象ルートのエラーをローカルで再現できる
- 根本原因をコードレベルで説明できる
- 修正後、対象ルートが本番相当実行で 200/正常描画になる
- 少なくとも `npm run build` と対象確認コマンドが通る

## Current hypothesis
- 共通レイアウト、認証境界、あるいはページ共通で読まれる server-side データ取得のどこかが壊れている可能性が高い
